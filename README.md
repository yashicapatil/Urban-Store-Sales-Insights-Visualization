Urban Store Sales Insights & Visualization

This project presents a detailed analysis of Urban Store’s annual sales using a structured retail dataset. The goal is to uncover insights about customer behavior, sales performance, product demand, and order trends through data cleaning, exploration, and visualization.

Project Overview:

The dataset contains 31,000+ records of customer orders with rich information such as:

Customer demographics

Order details

Payment and channel information

Shipping details

Product SKUs

Order status

Sales amounts

Multiple sheets in the Excel file provide aggregated summaries and visual dashboards.

Dataset Structure:
Main Dataset (Urban Store Sheet)

Contains 21 columns and ~31k rows:

Column Name	Description
Order ID	Unique identifier for each order
Cust ID	Customer identifier
Gender	Male/Female
Age, Age Group	Customer age details
Date, Month	Order date
Status	Delivered / Cancelled / Returned
Channel	Online platform (Amazon, Flipkart, etc.)
SKU	Product code
Category, Size, Qty	Product information
Currency	Transaction currency
Amount	Total bill amount
Ship-City, Ship-State, Ship-Postal-Code, Ship-Country	Shipping details
B2B	Whether the order is business-to-business

Additional Sheets & Insights:

Other sheets include prepared summaries for quick analysis:

Orders Channel – Sales split by e-commerce platform

Age vs Gender – Customer demographic patterns

Men vs Women – Gender-based purchase share

Order Status – Delivered vs Returned vs Cancelled

Top 5 – Best-selling categories/products

Sales vs Order – Monthly order and sales trend

Urban Store Report – Annual report summary

These sheets are helpful for building dashboards or presenting insights.

Objectives:

Understand customer demographics and purchasing behavior

Identify top-performing products and categories

Analyze sales trends across months and platforms

Track delivery performance and return/cancellation patterns

Create interactive and static visual dashboards

Tools Used:

Microsoft Excel – Data cleaning, pivot tables, charts

Python / Pandas (optional) – For data inspection and analysis

Power BI / Tableau (optional) – For advanced dashboarding
